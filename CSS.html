CSS

What is the Full form of CSS?

The full form of CSS is Cascading Style Sheets. It explains how to show HTML elements on display. For web designers, it is a powerful tool to adjust the design & control of web pages to demonstrate how they should be viewed. It is enabled by browsers and is specifically designed to distinguish the content of the document from the appearance of the report.

brief history of the CSS

  1. CSS was built in 1996 by the World Wide Web Consortium.

  2. The term cascading in CSS means that several style sheets may be added to a single web page.

  3. The extension used to save CSS files is “.css”.

Features of CSS

  1. In CSS, a global style sheet could be used such that with no trouble and taking fewer space users can perform any improvements in the style.

  2. One external style sheet can be produced and can be used for designing several websites.

  3. By only modifying one line, you can connect a different style sheet to a web page.

Benefits of CSS

  1. It enables you to describe a design for each feature of HTML. Each style can be applied to as many web pages as you like.

  2. You can easily configure the formatting of documents and keep different documents consistent.

  3. It enables formatting to be shared among several pages, that reduces the size of data transfer, helping to load pages faster.

  4. It provides independence from the web and supports new browsers as well.

  5. It has more features for representation than HTML. It enables you to boost the appearance of your Html files better than just using presentational features and properties of HTML
